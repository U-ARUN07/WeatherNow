## A Simple and Smart Weather Forecast WebApp

# WeatherNow

#### This web page is hosted [here!](https://U-ARUN07.github.io/WeatherNow)

### *Features*:

* Real-time weather updates using OpenWeatherMap API
* Displays accurate local date and time of the searched city
* Provides a 5-day weather forecast
* Automatically changes background based on weather condition and time
* Auto-refreshes weather data every 30 minutes
* Day and night theme switch
* Responsive UI for all devices
* Animated weather effects (clouds, rain, snow, thunder, fog, stars)
* User-friendly and visually appealing

### *Features to be added*:

* Option to save favorite cities
* Push notifications for extreme weather
* Multi-language support
* Progressive Web App (PWA) version for offline use
* Graphical temperature visualization
